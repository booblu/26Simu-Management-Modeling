# 🎓 26春-管理建模与仿真 (Management Modeling and Simulation)

欢迎来到本门课程的 GitHub 知识中枢！
本仓库是课程的“唯一真相源 (Single Source of Truth)”，所有的课件、大纲与实验材料将随课程进度在此处开源更新。

## 🗺️ 舱门导航 (Repository Directory)

为保证信噪比，本知识库采用高维结构化设计。请根据你需要探索的领域，进入对应的工作舱：

### 📖 [00_Syllabus_and_Docs](./00_Syllabus_and_Docs)
> **课程全局视界**。存放课程的元数据：教学大纲、学习路线图、排期规划以及必读书单。

### 👨‍🏫 [01_Lectures](./01_Lectures)
> **理论讲授殿堂 (Theory & Concepts)**。存放每周的授课核心理论讲义、关键概念抽象、数学推导及底层逻辑剖析（持续滚动更新）。

### 💻 [02_Labs_and_Workshops](./02_Labs_and_Workshops)
> **实操沙盘与工坊 (Practicum & Kickoff)**。存放每周的上机/沙盘实验指南、工程兵棋推演（Kickoff）简报、环境配置规范及红绿灯验收红线。

### 📝 [03_Assignments](./03_Assignments)
> **实验协议书**。存放每一次作业的官方说明、评分标准 (Rubrics) 与参考规范。

### 🛠️ [04_Resources](./04_Resources)
> **战略物资库**。存放辅助建模工具包、基准数据集以及推荐的开源代码库。

## 👩‍💻 学生作业提交引擎 (Student Interactive Node)

**【注意】这不仅是一个阅读库，更是一个双向交互系统。**

如果你是通过 GitHub Classroom 的邀请链接进入的：
1. 你的专属克隆分身中会拥有一个名为 **👉 `Student_Workspace`** 的文件夹。
2. 整个学期，请务必将你的所有作业代码及报告放置在 `Student_Workspace` 目录下。
3. 你分身的该目录具有**绝对隐私加密结界（Private）**，除导师外全维隐形。

有关详细的提交流程和 Git 操作规范，请查阅此空间内的指南。
