# 🎯 Project 1 交付极虎榜：Uniswap V3 底层状态机 (The Assignment Brief)

## 📌 商业背景与你的使命
作为这家微型量化对冲基金的 CTO，你的第一个任务是：**“造出一个坚不可摧的沙盒”**。
真实世界的去中心化交易所（如 Uniswap V3）是一个每秒钟承载千万美金的极其残酷的零和博弈场。在你让未来的 AI Agent 进去做主动市商（LP）或者套利之前，你必须在自己的本地机器上，1:1 像素级地还原这个交易所的物理法则。

**本周核心目标**：实现一个没有任何外部时间流逝、只接受“买/卖”离散事件指令的 V3 状态机仿真器。

---

## ⛰️ 最大可行圈定：MVP 必须实现的内容 (In Scope)

要拿到本周的基础分，你的 `src/simulator.py` 必须成功支撑以下三种绝对物理行为：

1. **池子初始化 (Pool Initialization)**
   - 能够设定初始价格 `sqrtPriceX96` 和初始流动性 `L`。
   - 能够初始化一个包含“抽屉”概念的 `Tick` 字典机制。
2. **同区间内交易与精准扣费 (Swap within a Tick)**
   - 给定任意一笔买单 $\Delta x$ 或卖单 $\Delta y$，能根据 $Q64.96$ 整数按位推导数学，精确算出池子价格的移动，以及用户应该拿回的确切数量。
   - 每次交易，必须按照 0.3%（或你定义的费率）把手续费留在系统内的 `feeGrowthGlobal` 变量中。
3. **跨区间连续吃单 (Crossing Ticks)**
   - 如果一笔巨额买单砸塌了当前的流动性深度，你的仿真器必须懂得：在这把椅子上扣除耗尽的流动性 -> 把秒针（`current_tick`）拨到下一把椅子 -> 注入下一个区间的流动性 -> 继续计算剩余的买单。

## 🚫 生死禁飞区：绝对不要做的事情 (Out of Scope)

时间是你在 Venture Studio 中最宝贵的资源。本周严禁在以下功能上浪费超过 1 分钟，否则必定导致项目烂尾：
- ❌ **不要写大盘大循环 (No Time Loop)**：P1 只是一个引擎，不需要写外部的 `while True` 来模拟时间的流逝。
- ❌ **不要接外部价格预言机 (No Oracles)**：不需要接 Binance API 去查价格，价格就是按单子砸出来的内部真实价格。
- ❌ **不要搞闪电贷或 MEV (No Flashbots)**：这个阶段只有你一个人在向池子发假请求，没有套利狗和你抢跑。
- ❌ **绝不允许用 `float` 糊弄计算**：见 `04_Resources/Project1_V3_Math_CheatSheet.md`，凡是被发现在核心函数里偷懒用了十进制导致挂掉红线断言的，直接 Fail。
